# node-3 FFmpeg media converter Deployment
## Overview

Native FFmpeg installation running on Windows 11 PRO .

---

# Environment

## Operating System
- Windows 11 Home 24H2

## Hardware
- Intel(R) Core(TM) i3-N300 (800 MHz)
- 8 GB RAM
- Intel QuickSync enabled
- 250 GB SSD

---

### Notes

- Chose a machine with an N300 Processor due to powerful encoding capabilities via Intel Quicksync and very low power consumption at peak utilization.
- Max power draw only ~29 watts
- Average of 60% performance improvement over the N100
- Used for more demanding task (2160p & 1080p encoding)
