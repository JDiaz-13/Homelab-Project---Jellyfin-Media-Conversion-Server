# node-3 FFmpeg media converter Deployment
## Overview

Native FFmpeg installation running on Windows 11 PRO .

---

# Environment

## Operating System
- Windows 11 Pro 24H2

## Hardware
- Intel(R) N100 (800 MHz)
- 16 GB RAM
- Intel QuickSync enabled
- 500 GB SSD

---

### Notes

- Chose a machine with an N100 Processor due to powerful encoding capabilities via Intel Quicksync and very low power consumption at peak utilization.
- Max power draw only ~27 watts
